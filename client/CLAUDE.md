# Мандри — Сервіс для зберігання даних про подорожі

## Контекст проєкту

Це дипломна робота з напряму **Інженерія програмного забезпечення**.
Тема: **«Сервіс для зберігання даних про подорожі»** з акцентом на внутрішній туризм в Україні.

Проєкт — React SPA (Single Page Application) без бекенду. Всі дані зберігаються в localStorage.
В дипломній роботі описується повна архітектура з Node.js + Express + PostgreSQL + Cloudinary,
але для демонстрації використовується фронтенд-версія.

## Технологічний стек

- **React 18** + Vite — фронтенд фреймворк
- **React Router v6** — маршрутизація
- **Tailwind CSS** — стилізація
- **Leaflet + React-Leaflet** — інтерактивні карти
- **Nominatim API** (OpenStreetMap) — геопошук міст
- **localStorage** — зберігання даних (замість PostgreSQL у демо)
- **Lucide React** — іконки

## Ключові функції сервісу

1. **Реєстрація / вхід** — з username для пошуку серед користувачів
2. **CRUD подорожей** — створення, перегляд, редагування, видалення
3. **Статуси** — Виконана / Планується / Зараз в дорозі / Скасована
4. **Видимість** — публічна або приватна подорож
5. **Конструктор сторінки** (BlockEditor) — блоки: текст, фото, карта, транспорт, житло
6. **Планувальник зупинок** (PolarPlanner) — міста з автодоповненням, транспорт між ними, розрахунок часу
7. **Планувальник по днях** (TripPlanner) — активності з типом, часом, ціною
8. **Бюджет** — автоматичний підрахунок витрат
9. **Карта маршруту** — Leaflet + OpenStreetMap, клік для додавання точок
10. **Карта регіонів України** — SVG з відвіданими/запланованими областями
11. **Стрічка** (Explore) — публічні подорожі + пошук користувачів
12. **Профіль** — статистика, карта мандрів, список подорожей з бюджетом

## Правила розробки

- Мова інтерфейсу: **українська**
- Стиль: Tailwind utility classes, кастомні токени в `tailwind.config.js`
- Кольори: brand (зелений) + earth (охра) + stone (нейтральний)
- Шрифти: Playfair Display (заголовки) + Nunito (тіло)
- Компоненти без PropTypes — проєкт не використовує TypeScript
- Leaflet конфігурується один раз через `src/utils/leafletSetup.js`
- Всі кнопки у формах мають `type="button"` щоб не тригерити submit
- Стан форм зберігається окремо від глобального стану (useCallback для onChange)

## Поточні відомі проблеми

- SVG карта України (`UkraineMap.jsx`) — форми областей спрощені, не відповідають реальним
- При великому обсязі фото (base64) localStorage може переповнюватись

## Що НЕ треба змінювати

- Загальна структура папок
- Назви компонентів (вони описані в дипломній роботі)
- Стек технологій
- Українська мова інтерфейсу
